# MA505GroupProject

### Purpose
This project is meant to find the causal effect of various factors on aircraft accident fatalities.

### Files
`data_labeling.py` is a tool to classify the unstructured data from the dataset

`evalute_classifer.py` includes tools to validate the data labeler from a manually made dataset

`regressions.py` includes all the tools for finding the graphs, and making linear regressions.
